Features

Built for AI-First Development

LCP provides high-fidelity context about libraries so AI systems can reason about APIs accurately without parsing source code at runtime.

Symbol-Centric

Everything revolves around well-defined symbols with stable identifiers that don't change across refactors.

Language-Agnostic

Works across Python, JavaScript, TypeScript, Java, C#, Rust, Go, and more. One format for all ecosystems.

Tool-Friendly

Easy to diff, index, validate, and generate. Designed for integration with IDEs, linters, and documentation tools.

Semantic Rich

Captures not just signatures, but effects, stability, deprecations, and behavior semantics.

Extensible

Vendor or ecosystem-specific data via x-* fields. Your custom metadata, our standard format.

Validated

JSON Schema ensures correctness. Every LCP document can be validated against the official spec.

Use Cases

Powering the Future of Development

AI Code Assistants

Feed LLMs accurate, structured library context to generate correct API calls without hallucinations. LCP provides the ground truth AI needs.

Documentation Generators

Generate beautiful, consistent documentation from LCP files. All the semantic richness you need in one portable format.

IDE Integration

Provide rich autocomplete, inline documentation, and intelligent suggestions using pre-computed library context.
