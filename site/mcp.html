The Bridge

What is MCP?

Model Context Protocol (MCP) is an open standard for connecting AI assistants to external data sources and tools.

LCP + MCP = Perfect Match

LCP provides the structured library data. MCP provides the communication protocol. Together, they enable AI agents to access accurate API information on-demand.

  • Version-accurate signatures

Get Started

Quick Start

Set up the LCP-MCP server in under 5 minutes.

1 Install & Generate LCP

bash 
# Install the SDK
pip install lcp-python-sdk

# Generate LCP for your libraries
lcp-python scan requests -o requests.lcp.json
lcp-python scan pandas -o pandas.lcp.json

2 Start the MCP Server

bash 
# Start MCP server for a single library
lcp-python serve requests.lcp.json

# With a custom name
lcp-python serve requests.lcp.json --name requests-api

3 Configure Your AI Client

Add the server to your MCP client configuration. Here's an example for Claude Desktop:

json 
{
  "mcpServers": {
    "requests-api": {
      "command": "lcp-python",
      "args": ["serve", "/path/to/requests.lcp.json"]
    },
    "pandas-api": {
      "command": "lcp-python",
      "args": ["serve", "/path/to/pandas.lcp.json"]
    }
  }
}

Config file locations:
macOS: ~/Library/Application Support/Claude/claude_desktop_config.json
Windows: %APPDATA%\Claude\claude_desktop_config.json

API Reference

Available MCP Tools

The LCP-MCP server exposes these tools for AI agents to query library information.

get_manifest
Get library metadata including name, version, language, and distribution information.
Returns:
{ name, version, language, distribution }
list_modules
List all modules in the library. Useful for discovering available submodules.
Returns:
["requests", "requests.api", "requests.models"]
list_symbols
Browse symbols with optional filtering by module or kind.
module string (optional)
kind "function" | "class" | "method" | ...
get_symbol
Get full details for a specific symbol including signatures, semantics, and effects.
symbol_id string (required)
Example: "requests:get", "pandas:DataFrame#merge"
search_symbols
Find symbols by text search across names, summaries, and descriptions.
query string (required)
limit number (default: 10)
get_class_members
Get all methods and attributes of a class. Essential for understanding class APIs.
class_id string (required)
Example: "requests:Response", "pandas:DataFrame"

Programmatic Usage

You can also create and customize the MCP server programmatically:

python 
from lcp_python_sdk.mcp_server import create_server, run_server

# Create a customized server
server = create_server(
    "path/to/manifest.lcp.json",
    name="my-library-api"
)

# Or run directly with defaults
run_server("path/to/manifest.lcp.json")

# You can also load multiple manifests
from lcp_python_sdk import scan

# Generate and serve on-the-fly
lcp_doc = scan("requests")
lcp_doc.to_file("/tmp/requests.lcp.json")
run_server("/tmp/requests.lcp.json")
